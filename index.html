      Architecture and Flow

      System Architecture

      Frontend HTML, CSS, and JavaScript maintain the cube UI. Flask receives the cube state and executes the C++ solver. The C++ layer validates state, explores graph nodes, and returns JSON solution steps.

      Data Structures

      Queue, stack, arrays, sets, hash maps, graph state nodes, and recursive search trees are used across the solver modules.

      Real-World Applications

      Robotics path planning, game AI, network routing, logistics optimization, puzzle solving, and decision support systems use the same search and pruning ideas.
